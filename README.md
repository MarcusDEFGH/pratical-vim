# Pratical Vim and Neovim

# Commands

## Modo Normal

* cmd: `dib` or `di(` ou `di)`
 - (word1 word2) -> ()

* cmd: `di[` or `di]`
 - [word1 word2] -> []

* cmd: `diB` or `di{` or `di}`
 - {word1 word2} -> {}

* cmd: `di\``
 - \`word1 word2\` -> \`\`

* cmd: `di'`
 - 'word1 word2' -> ''

* cmd: `di'`
 - 'word1 word2' -> ''

* cmd: `di"`
 - "word1 word2" -> ""

* cmd: `dit`
 - <div>Delete inner Tag</div> -> <div></div>
